File tkcm.c
implements algorithm TKCM


File paper.c
contains the running example from the paper and can be
executed with: `make paper && ./paper`
The output contains the assumed missing value in the first column and the
imputed value in the second column


File chlorine.c
contains an example that imputes a missing value in the
Chlorine dataset. It can be executed with: `make chlorine && ./chlorine`
The output contains the assumed missing value in the first column and the
imputed value in the second column


File cl2fullLarge.dat
contains the Chlorine dataset and was obtained from:
https://www.cs.cmu.edu/afs/cs/project/spirit-1/www/#Download
Please acknowledge the original dataset providers if you intend to
use this dataset.
